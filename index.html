Visualisation de données issues de bases MongoDB

L'étude de ce projet a pour but de visualiser 3 bases de données différentes interrogées via des requêtes MongoDB.

Voir les visualisations

Mise en oeuvre de l'étude

Les bases MongoDB permettent des études précises et consises des données via différents langages tels que R ou encore Python.

Le cadre de notre étude est de visualiser certaines modalités de différentes bases MongoDB utilisant ces deux langages.

Voir les bases de données

Bases MongoDB

  • Centres de vaccination Covid19
  • Publications scientifiques
  • Données issues de NYfood

La base doctolib contient des informations, récupérées en janvier 2022, relatives aux centres de vaccination contre la Covid19 en Bretagne.


  • 5 éléments identifiant un individu

    "_id" : identifiant unique permettant d'identifier la personne venant au centre de vaccination

    "name" : nom du centre de vaccination

    "url_doctolib" : url du site doctolib utilisé

    "location" : identifie la longitude et la latitude du centre de vaccination

    "visit_motives" : raisons de la visite (type de vaccination, 1ère injection, etc)

  • Visualisation : via R/Python

    Avec R/Python depuis cette base, nous allons générer une carte des centres de vaccination situés à moins de 50km de Rennes.

La base publications contient les informations relatives aux publications de scientifiques du laboratoire IRISA pour l’année 2021 (extraites du service HAL).


  • 6 éléments identifiant une publication scientifique

    "_id" : identifiant unique permettant d'identifier la publication scientifique

    "halld" : ??

    "uri" : url du site ??

    "title" : titre de la publication

    "conferenceTitle" : nom de la conférence où ??

    "authors" : nom des différents auteurs de la publication

  • Visualisation : via R/Python

    Avec R/python, nous allons repérer les liens entre les auteurs de ces publications, en les distinguant par leurs nombres de publications et en représentant les liens (co-publications) existant entre les auteurs. On se focalisera sur les 20 auteurs les plus prolifiques (i.e. qui ont participé à l’écriture du plus grand nombre d’articles).

La base food contient des informations relatives à des restaurants de New-York.


  • 7 éléments identifiant un restaurant de New-York

    "_id" : identifiant unique du restaurant dans la base

    "address" : localise le restaurant (coordonnées gps, rue, numéro du bâtiment, code zip)

    "borough" : quartier où se situe le restaurant

    "cuisine" : type de cuisine du restaurant

    "grades" : notes données au restaurant avec la date ainsi que le score attribué

    "name" : nom du restaurant

    "restaurant_id" : identifiant du restaurant
